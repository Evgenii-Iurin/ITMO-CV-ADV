ITMO | COURSE | Computer Vision Advanced

# HW 1

### Prepare repository

**Download data**
1. Download dataset from `https://disk.yandex.ru/d/gJlyi4A2Bk178A`
2. Add to the folder `src/hw_1`
3. Unzip with command `unzip src/hw_1/data.zip -d src/hw_1/data`

**Install  environment**
1. Make sure you already have Poetry installed on your system
2. Make sure you use Python3.12
3. `poetry install`
4. `poetry shell`
5. `pre-commit install`
